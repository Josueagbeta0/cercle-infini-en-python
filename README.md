# Cercle infini en Python

Ce projet utilise la bibliothèque Turtle pour dessiner un motif de cercle qui se répète en boucle.

## Description

Le script affiche un motif visuel en forme de cercle en continu. Il réinitialise le dessin à chaque cycle pour créer une répétition sans fin.

## Prérequis

- Python 3.x
- Bibliothèque Turtle (inclus avec Python)

## Exécution

Depuis le dossier du projet, exécutez :

```bash
python cercle.py
```

## Fonctionnement

- Le programme dessine un motif de cercle avec des rotations et des déplacements.
- La boucle principale répète l’affichage à l’infini.
- Pour arrêter l’exécution, fermez la fenêtre Turtle.

## Notes

Si vous souhaitez modifier les couleurs ou la vitesse, modifiez les lignes suivantes dans le fichier `cercle.py` :

- `color("cyan")`
- `bgcolor("black")`
- `speed(11)`
